Selected Works

A collection of my recent projects and contributions.

  • Codebase Tattoo Studio

    A command-line tool that generates customizable ASCII “signature” banners from text or a file, and can prepend them recursively to code files in a directory.

  • Translate README's Readily

    A utility that helps translate or adapt README files into multiple languages to make project documentation more accessible to international users.

  • Zooplankton Imaging Tool

    Video flattening library to document plankton locomotion.

  • Vacuum Pump Engineering Library

    A Python/C++ library for vacuum theory and pump-system calculations, built to turn textbook equations into a solver for many variable permutations.

  • Python Posterize Library

    SAn image-processing helper that uses “magic wand” style region selection to enhance, recolor, or stylize posterized or vector-like images.

  • Useful Developer Aliases

    A curated collection of shell or Git command aliases and shortcuts designed to speed up everyday development and repository workflows.

  • Multilingual GIF Creator

    A tool for generating GIFs with multilingual text overlays, aimed at making shareable animations that can include captions in several languages.

  • Physical Menu to Website Script

    A script that helps convert the contents of a physical or scanned restaurant/menu into structured data suitable for putting onto a website menu page.
