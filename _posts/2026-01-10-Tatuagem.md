Tatuagem (Portuguese for "tattoo") v.0.1.4 is done! This was a fun collaboration between me and [@DerekDickerson](https://github.com/DerekITCoder). You can find the project on PyPI at [https://pypi.org/project/tatuagem/](https://pypi.org/project/tatuagem/) and on [tatuagem github](https://github.com/juleshenry/tatuagem/)

Fun little thing to stamp text on projects recursively. Built in Python.

### Usage

```bash
usage: tatuagem [-h] [--text TEXT] [--backsplash BACKSPLASH] [--font FONT]
                [--pattern PATTERN] [--margin MARGIN] [--recurse-path RECURSE_PATH]
                [--file FILE] [--overwrite]

Tatuagem

options:
  -h, --help            show this help message and exit
  --text TEXT           Set the text
  --backsplash BACKSPLASH
                        Choose backsplash
  --font FONT           Set the font
  --pattern PATTERN     Set the pattern for backsplash
  --margin MARGIN       Margin top and bottom for text
  --recurse-path RECURSE_PATH
                        Path to recurse and apply tattoo
  --file FILE, -f FILE  Read text from file
  --overwrite           Overwrite existing tattoos in files
```

To use it, you can simply run it with the text you want to "tattoo" onto your files:

```bash
tatuagem --text "CONFIDENTIAL" --recurse-path ./src
```
